# Problem Statement
It is very easy to lose track of where your money goes. A lot of people dont know how much they spend on small things like coffee or snacks. This project solves that by providing a simple tracker in the command line.

# Scope
The goal is to build a Python console app to track personal expenses. It is designed to be simple and good for beginners. It runs in the terminal and does not reqiure any special software besides Python.

# Target Users
This tool is for students, freelancers, or anyone who wants to watch their buget closely. You don't need advanced tech skills to use it, just basic knowledge of running a script.

# High-level Features
Record expenses with amount, category, and date
View a list of all recorded items
Show total spending and spending by catagory.
