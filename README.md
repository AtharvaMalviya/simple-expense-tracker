# simple-expense-tracker
## Overview
This is a simple console-based personal expense tracker written in Python. It allows users to add, view and to see spending summaries.

## Features
add a new expense (date, category, description, amount)
View a table of all your past expenses
See a summary of total money spent
Group spending by categories

## Technology Used
Python 3 (no extra libaries needed)

## How to Run
Make sure you have Python installed on your computer.
Copy or download this repositery.
open a terminal or go to the project folder.
Run the program command: python main.py

## Usage Example
After you start the program, you will see a menu like this:
Add expense
View expenses
View summary
Exit
Just type the number you want. For example, if you chose 1, it will ask for the date and price.

## Testing
